# Daily Deli – E-Commerce Food Platform

Daily Deli is a full-stack e-commerce web application developed by my team and I. The platform is designed to combine grocery shopping with ready-made meals, dietary filtering, personalised recommendations, and weekly meal planning.

The system supports both customer and administrator functionality. Customers can register, manage profile-based dietary preferences, receive recipe suggestions, plan meals, earn loyalty rewards, and interact with a community-driven blog platform. Administrators can manage inventory, verify blog posts, track orders, adjust recipes, and apply discounts.

---

## Project Overview

Daily Deli was created to provide users with a more personalised and convenient online shopping experience. Instead of only offering grocery products, the platform also supports ready-made meals, recipe recommendations, dietary preferences, allergen awareness, and sustainable local product sourcing.

The project helped strengthen my skills in full-stack development, UX planning, data modelling, and building scalable, user-centered web systems.

## Features

**Customer Features:** User registration | Dietary preference profiles | Vegetarian and vegan filtering | Recipe suggestions | Weekly meal planning | Loyalty rewards | Nutritional information | Allergen alerts | Community blog platform | Curated recipe cross-selling
